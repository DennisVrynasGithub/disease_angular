Diseases

Our database contains over 3900 diseases that we can find. For each disease we have to find the definition of the disease, the cause (s) that cause the disease, the symptoms that the each and finally how to deal with the illness.

Symptoms

The user can choose a possible symptom and accordingly the symptom he has stated will find possible illnesses that have these symptoms In addition to the name of the possible diseases, he will find possible treatments.

Web-app Disease-Symptom

Application Disease-Symptom is an online application that helps us find any disease and learn about it. We can find the definition of the disease, the cause / s that cause the disease, the symptoms each one and finally the way of dealing with the illness. Otherwise for more details you have to visit your doctor!!!!

Questionnaires

The application contains 6 questionnaires. Each of these refers to different types of diseases. The questionnaire is stored in our database so that we can manage user responses.

Android

The web application is also available on mobile and tablets. The application is called disease_symptom_app and has exactly the same capabilities as the web application.

About us

In both the Android and Android app, the user must never believe that the application replaces the doctor. It is an application that provides information. For any health issue VISIT YOUR DOCTOR .
